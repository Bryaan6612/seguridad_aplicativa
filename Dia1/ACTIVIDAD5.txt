¿Qué es una red?
Una red es un enlace entre uno o más dispositivos los cuales se pueden comunicar y realizar transferencia de datos.

¿Qué es el modelo TCP/IP?
Es el modelo el cual permite la asignación y configuración de las direcciones para cada dispositivo cuando esta en uan red

¿Qué es una IP?
El segmento asignado a un dispositivo dentro de una red y por cual permite la comunicación y transferencia de datos

¿Qué es el protoclo TCP?
Es el que permite la comunicación del servidor y de dispositivos entre ellos

Dentro del protocolo TCP ¿Qué es el protocolo HTTP?
Es el que valida, permite la comunicación web entre el servidor y los dispositivos

Puerto del protocolo HTTP
44

Puerto del protocolo HTTPS
400

Diferencia entre HTTP y HTTPS
La seguridad y encriptamiento de la información entre ambos protocolos, es más seguro HTTPS

¿Qué significa el modelo CLiente-Servidor?
El cliente manda solicitudes/peticiones al servidor y el servidor atiende las peticiones y dandole una respuesta al cliente

¿Qué es una petición?
Una petición es la solictud de datos/información al servidor 

¿Qué es una response?
La solicitud del cliente devuelta por el servidor con lo solicitado

Secciones de una petición
-validación
-conexión con el server

Secciones de una respuesta
- Conexión con el cliente
- Transferencia de la request

¿El código de respuesta 4004 y 500 a que se refiere?
El 4004 a que no puede encontrar la dirección/archivo en el servidor y el 500 es que no se tienen los permisos para acceder al directorio dentro del servidor
